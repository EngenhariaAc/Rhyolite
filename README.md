# 🌋 Rhyolite Project

Welcome!

This repository explores rhyolite, a fine-grained volcanic rock rich in silica.

## Features
- Rhyolite properties
- Formation simulation
- Geological notes
- Sample data

## About Rhyolite
Rhyolite forms from highly viscous lava that cools quickly at or near the Earth's surface.

A fascinating rock born from explosive volcanic activity.
